Queries 

self.db['entry'].aggregate(
  [{
    '$match': {
      'author_id': {
        '$ne': None,
        '$exists': True
       }
     }
    },
    {
      '$lookup': {
        'from': 'author',
        'localField': 'author_id',
        'foreignField': 'id',
        'as': 'author'
      }
    },
    {
      '$unwind': '$author'
    },
    {
      '$lookup': {
        'from': 'blog',
        'localField': 'blog_id',
        'foreignField': 'id',
        'as': 'blog'
      }
    },
    {
      '$unwind': {
        'path': '$blog',
        'preserveNullAndEmptyArrays': True
      }
    },
    {
      '$addFields': {
        'blog': {
          '$ifNull': ['$blog', {
            'id': None,
            'title': None
          }]
        }
      }
    },
    {
      '$match': {
        'author.name': {
          '$eq': 'Paul'
        }
      }
    }, 
    {
      '$project': {
        'id': True,
        'blog_id': True,
        'author_id': True,
        'content': True,
        'blog.id': True,
        'blog.title': True
      }
    }]

qs = Entry.objects.filter(author__name='Paul').select_related('blog')

Djongo generates complex, error free, aggregation queries automatically.

It takes the query on the top and automatically generates the pymongo query document as the output.
